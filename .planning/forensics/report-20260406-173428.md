## Recommendations

### Immediate: Start Fresh with `npm create convex@latest`

The manual scaffolding introduced enough issues (wrong package name, loose version pins, phantom deps, committed node_modules) that patching is more work than restarting. Use Convex's official scaffolder:

```bash
npm create convex@latest calmdo
```

This generates: proper `.gitignore`, correct Convex wiring, shadcn/ui pre-configured, no phantom dependencies.

### Process: Fix the Three Systemic Issues

1. **Research must check quickstart/getting-started pages first** — before diving into individual feature docs, the research step should always check the platform's canonical setup instructions.

2. **SUMMARY.md generation must be atomic with code commit** — or a post-execution health check must detect and repair missing summaries before any downstream workflow runs.

3. **`/gsd:next` must delegate, never improvise** — add a hard constraint that `/gsd:next` is a router only. It determines the next command and invokes it. It never performs the command's work inline.
